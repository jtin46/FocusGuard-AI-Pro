# 🛡️ FocusGuard AI Pro
A sophisticated AI-powered study monitor that ensures you stay awake and focused during deep work sessions.

## ✨ Features
- **High-Accuracy Neural Tracking**: Uses SSD Mobilenet V1 for precise eye landmarking.
- **Drowsiness Logic**: Industry-standard EAR (Eye Aspect Ratio) calculation to detect sleep.
- **Focus Dashboard**: Real-time session stopwatch and distraction counter.
- **Professional UI**: Modern dark-mode Glassmorphism design.

## 🚀 Quick Start
1. Clone this repo.
2. Add your custom audio to `assets/alert.mp3`.
3. Open `index.html` with Live Server.

## 🛠️ Built With
